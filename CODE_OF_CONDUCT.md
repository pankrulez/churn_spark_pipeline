# Code of Conduct

## Purpose

This project is intended to be a respectful, inclusive, and professional learning environment.  
All participants—contributors, reviewers, and maintainers—are expected to follow this Code of Conduct when engaging with the project or its community.

This document defines expected behavior and outlines how unacceptable conduct is handled.

## Expected Behavior

Participants are expected to:

- Communicate respectfully and professionally
- Be considerate of differing viewpoints, backgrounds, and experience levels
- Provide constructive feedback focused on ideas and code, not individuals
- Accept feedback gracefully and in good faith
- Act with integrity and honesty in all interactions
- Follow project guidelines, policies, and decisions made by maintainers

## Unacceptable Behavior

The following behavior is not tolerated:

- Harassment, discrimination, or personal attacks of any kind
- Insults, derogatory comments, or inflammatory language
- Dismissive, hostile, or deliberately disruptive behavior
- Publishing private information without consent
- Repeatedly ignoring project guidelines or maintainer instructions
- Any conduct that creates an unsafe or unproductive environment

## Scope

This Code of Conduct applies to all project-related spaces, including but not limited to:
- GitHub issues and pull requests
- Code reviews and discussions
- Project documentation and communication channels
- Any interaction representing or involving this project

## Reporting Issues

If you experience or witness unacceptable behavior, report it responsibly.

❌ Do not engage in public arguments or retaliation.

✅ Report the issue privately by:
- Contacting the repository owner via their GitHub profile  
- Using a private communication channel if one is explicitly provided

Reports should include:
- A clear description of the incident
- Relevant context, links, or screenshots if applicable
- The names or usernames involved (if known)

All reports will be handled with discretion.

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Actions may include:
- Informal warnings
- Requests for corrective action
- Temporary or permanent bans from participation
- Removal of content that violates this policy

Decisions made by maintainers are final.

## No Retaliation

Retaliation against anyone who reports a concern in good faith is strictly prohibited and will be treated as a violation of this Code of Conduct.

## Acknowledgement

By participating in this project, you agree to abide by this Code of Conduct and accept enforcement actions when necessary.

---

This Code of Conduct exists to protect the quality of the project and the people contributing to it.
